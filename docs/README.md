# Daily Sales Dashboard — Management View (v3)

This is a static dashboard designed to run:
- locally (double-click `index.html`)
- on GitHub Pages

## Included views
- KPI strip (Sales, Budget, Achievement, Variance, Qty, Bonus share)
- Sales vs Budget by month
- Team ranking (Top 10)
- Top items (Top 10)
- Variance waterfall (Budget → Actual) driven by Product Groups
- Full detail table + CSV export

## Publish on GitHub Pages
1. Upload all files to repo root
2. Settings → Pages → Deploy from branch → `main` → `/(root)`
3. Open the generated link
